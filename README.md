# Prodigy InfoTech Machine Learning Internship

This repository contains the five tasks assigned during my Machine Learning internship at Prodigy InfoTech. Each task demonstrates a different area of ML, from basic regression to deep learning.

## 📂 Project Structure

### [Task 01: House Price Prediction](https://github.com/YourUsername/Prodigy_ML_Internship/tree/main/Task_01)
* **Goal:** Predict house prices based on square footage, bedrooms, and bathrooms.
* **Algorithm:** Linear Regression.
* **Library:** Scikit-learn, Pandas.

### [Task 02: Customer Segmentation](https://github.com/YourUsername/Prodigy_ML_Internship/tree/main/Task_02)
* **Goal:** Group retail store customers based on purchase history and income.
* **Algorithm:** K-Means Clustering.
* **Key Concept:** The Elbow Method for optimal K.

### [Task 03: Cat vs Dog Image Classification](https://github.com/YourUsername/Prodigy_ML_Internship/tree/main/Task_03)
* **Goal:** Classify images of cats and dogs.
* **Algorithm:** Support Vector Machine (SVM).
* **Preprocessing:** Grayscale conversion and pixel normalization.

### [Task 04: Hand Gesture Recognition](https://github.com/YourUsername/Prodigy_ML_Internship/tree/main/Task_04)
* **Goal:** Identify 10 different hand gestures from near-infrared images.
* **Algorithm:** Convolutional Neural Network (CNN).
* **Library:** TensorFlow/Keras.

### [Task 05: Food Recognition & Calorie Estimation](https://github.com/YourUsername/Prodigy_ML_Internship/tree/main/Task_05)
* **Goal:** Identify food items and map them to nutritional data.
* **Algorithm:** Transfer Learning using **MobileNetV2**.
* **Dataset:** Food-101.
